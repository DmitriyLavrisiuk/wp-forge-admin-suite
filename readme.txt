Short admin panel plugin scaffold with a React UI.

== Description ==
Provides an admin-only React application scaffold with a REST status endpoint.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
